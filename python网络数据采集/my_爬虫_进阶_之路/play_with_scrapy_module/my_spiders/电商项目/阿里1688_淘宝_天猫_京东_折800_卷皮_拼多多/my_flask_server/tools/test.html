12.12供暖季，别让暖气病找上你

夏天吹空调过多容易引发空调病大家都知道，可是冬天在有暖气的室内呆太久也会引发暖气病你知道吗？

暖气病不是单指一种病，是指因为长时间呆在暖气室内引发的一系列疾病，北方冬季室内暖气开放，门窗紧闭就会导致空气流通不畅、干燥，引起人体免疫功能和抵抗力的下降，导致人们口干舌燥、头晕眼花、软弱无力等，这些症状医学上称为“暖气病”。

暖气病的罪魁祸首是室内空气干燥，浑浊，干燥浑浊的环境容易让人的呼吸系统抵抗力降低，浑浊的空气更是病毒滋生的温床，所以要对付暖气病只需要从以下这两个方面下手即可：


增加空气湿度，可以在室内摆几盆水，如果觉得太麻烦，最快速有效的办法是使用加湿器，它不仅能增加空气的湿度，在加湿器里加几滴醋，还能起到杀菌作用的，湿润的空气还能提高人们的睡眠质量。

要解决室内空气浑浊的问题，除了开窗通风，养一些绿色植物之外，还可以买一台空气净化器，空气净化器能够吸附、分解或转化各种空气污染物，可以在很短的时间内让室内空气焕然一新，有效提高空气清洁度，去除细菌和异味，让你在温暖的室内也能呼吸洁净的空气。
